개인정보 처리방침

Buylink 견적 분석기 Chrome 확장 프로그램

Buylink(이하 "서비스")는 사용자의 개인정보를 소중히 여기며, 관련 법령을 준수합니다. 본 개인정보 처리방침은 Chrome 확장 프로그램 "Buylink 견적 분석기"의 개인정보 처리에 관한 사항을 안내합니다.

1. 수집하는 정보

본 확장 프로그램은 다음의 정보에 접근합니다:

  • 현재 탭 URL: 메루카리(Mercari) 상품 페이지 URL을 읽어 상품 정보를 분석합니다.
  • 상품 정보: 상품명, 가격, 카테고리, 이미지 URL 등 공개된 상품 정보를 수집합니다.
중요: 본 확장 프로그램은 사용자의 개인 식별 정보(이름, 이메일, 결제 정보 등)를 수집하지 않습니다.

2. 정보의 이용 목적

수집된 정보는 다음의 목적으로만 사용됩니다:

  • 메루카리 상품의 무게 및 부피 예측
  • 해외배송 견적 계산
  • 사용자에게 예상 비용 정보 제공

3. 정보의 저장 및 보관

  • 본 확장 프로그램은 사용자의 브라우저에 어떠한 개인정보도 저장하지 않습니다.
  • 분석 요청 시 상품 정보는 서버로 전송되어 처리된 후, 별도로 저장되지 않습니다.
  • 모든 분석은 실시간으로 처리되며, 세션 종료 후 데이터는 삭제됩니다.

4. 정보의 제3자 제공

본 서비스는 수집된 정보를 제3자에게 판매, 대여 또는 공유하지 않습니다. 단, 다음의 경우는 예외로 합니다:

  • 법령에 의해 요구되는 경우
  • 사용자의 명시적 동의가 있는 경우

5. 외부 서비스 연동

본 확장 프로그램은 다음의 외부 서비스와 연동됩니다:

  • Buylink API 서버: 상품 정보 크롤링 및 견적 계산
  • 메루카리(Mercari): 상품 페이지 접근 (공개 정보만 수집)

6. 사용자의 권리

사용자는 언제든지 다음의 권리를 행사할 수 있습니다:

  • 확장 프로그램 비활성화 또는 삭제
  • 브라우저 설정을 통한 권한 관리
  • 개인정보 관련 문의 및 요청

7. 보안

본 서비스는 사용자 정보 보호를 위해 다음의 보안 조치를 시행합니다:

  • HTTPS를 통한 암호화된 통신
  • 최소한의 권한만 요청
  • 정기적인 보안 점검

8. 개인정보 처리방침의 변경

본 개인정보 처리방침은 법령 또는 서비스 변경에 따라 수정될 수 있습니다. 변경 시 확장 프로그램 업데이트 또는 본 페이지를 통해 공지합니다.

9. 문의

개인정보 처리에 관한 문의사항이 있으시면 아래로 연락해 주세요:

  • 서비스명: Buylink
